# SaiyoniX

**Engineering Transformative Evolution**

SaiyoniX is a technology-driven collective inspired by the Manipuri word **Saiyon**,
representing transformation, evolution, and disruptive growth.

## What This Repository Contains
- Official SaiyoniX website
- Built with Next.js (App Router) and Tailwind CSS v4
- Clean, scalable architecture for long-term growth

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS v4
- PostCSS

## Project Status
- ✅ Phase 1: Foundation, branding, and timeline (completed)
- ⏭ Phase 2: Motion, projects, and brand expansion (upcoming)
